# Assignment 41 :

This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Write a recursive program which Accept number from user and Display below pattern.
2. Write a recursive program which Accept number from user and return simmation of its digits.
3. Write a recursive program which Accept number from user and count number of character.
4. Write a recursive program which Accept number from user and return its factorial.
5. Write a recursive program which Accept number from user and its product of digits.

Commands used to execute these codes are:-

g++ filename.c -o myexe myexe
